## 微積

- $y=f(x)$ が $x=a$ で微分可能なとき、$(a,f(a))$ における接線は $y = f'(a)(x-a)+f(a)$
- $(f(x)+g(x))' = f'(x)+g'(x)$
- $(cf(x))' = cf'(x)$
- $(f(x)g(x))' = f'(x)g(x)+f(x)g'(x)$
  - 証明には $\dfrac{f(x)g(x)-f(a)g(a)}{x-a} = \dfrac{f(x)-f(a)}{x-a} g(x) + f(a) \dfrac{g(x)-g(a)}{x-a}$ を用いる。
- $(x^n)' = nx^{n-1}　(n∈\mathbb{R})$
